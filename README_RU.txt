Burrata Calculator Web v1.12.4

Это веб-версия калькулятора на основе BurrataCalculator_UI_v1_12_4_BowlLimit.zip.

Что внутри:
- вкладка «Буррата»;
- вкладка «Трюфель»;
- вкладка «Чан»;
- настройки формул с сохранением в браузере;
- округление 0,6 как в Android-версии;
- исправленное деление молока по партиям: сумма партий совпадает с общим молоком;
- строгий лимит начинки с потерями в одном тазу: классика не больше 3000 г, трюфель не больше 2067 г или значения из настроек;
- для трёхзначной страчителлы при расчёте сливок берётся первая цифра;
- PWA: можно добавить на главный экран телефона;
- иконка бурраты;
- GitHub Pages workflow.

Как открыть без публикации:
1. Распакуйте архив.
2. Откройте index.html в браузере.

Как выложить на GitHub Pages:
1. Создайте новый репозиторий, например BurrataCalculatorWeb.
2. Загрузите все файлы из этой папки в корень репозитория.
3. Откройте Settings → Pages.
4. В Source выберите GitHub Actions.
5. Перейдите во вкладку Actions и запустите Deploy web app to GitHub Pages.
6. После успешной сборки сайт будет доступен по ссылке вида:
   https://nkiyanov68-source.github.io/BurrataCalculatorWeb/

Как добавить на экран телефона:
- iPhone: открыть ссылку в Safari → Поделиться → На экран «Домой».
- Android: открыть ссылку в Chrome → ⋮ → Добавить на главный экран.
